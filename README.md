# Blue Chip Engineering Website

Static website for `bluechipspc.com`, based on CAPSA Soft's current content structure and adapted for Blue Chip Engineering branding.

## Files

- `index.html` - single-page website
- `styles.css` - site styling
- `script.js` - mobile navigation
- `assets/` - optimized logo and favicon assets
- `CNAME` - GitHub Pages custom domain

## GitHub Pages

After pushing to GitHub, enable Pages from the repository settings:

1. Source: Deploy from branch
2. Branch: `main`
3. Folder: `/root`
4. Custom domain: `bluechipspc.com`
